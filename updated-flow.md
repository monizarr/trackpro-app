## Update Flow batch produksi terbaru 

Setelah hasil potong pada batch produksi diinput dan terverifikasi oleh kepala produksi.
Tahap selanjutnya yaitu assign ke penjahit, saya ingin memecah batch menjadi sub batch untuk di assign ke masing masing penjahit 

1. kepala produksi verifikasi potongan
2. kepala produksi assign ke beberapa penjahit, 
    2.1. masing masing assign ke panjahit ini menjadi sub batch, 
    2.2. kepala produksi dapat menginputkan hasil jahitan di setiap sub batch
    2.3. kepala produksi konfirmasi sub batch 
    2.4. kepala produksi assign sub batch ke finishing
    2.5. kepala produksi konfirmasi finishing sub batch
    2.6. kepala produksi menyerahkan sub batch ke gudang
3. jika sub batch sudah selesai semua maka kepala produksi konfirmasi penyelesaian batch produksi
